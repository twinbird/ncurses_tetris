# curses tetris

A tetris clone in terminal.

![result](https://github.com/twinbird/ncurses_tetris/blob/master/play.gif?raw=true)

## build & execute

### Debian (ubuntu)

```
$ sudo apt install libncurses5-dev
$ make
$ ./tetris color
```

### CentOS

```
$ sudo yum install ncurses-devel
$ make
$ ./tetris color
```

### MacOS X

```
$ make
$ ./tetris color
```
